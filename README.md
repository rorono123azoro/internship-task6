# Internship Task 6: User Management System

A full-stack User Management System enabling CRUD operations on user profiles, connecting a React frontend with a Node.js/Express backend.

## Structure
- `/frontend`: React/Vite-based user interface
- `/backend`: Node.js Express server with database integration

## Features
- End-to-end functionality (frontend to backend communication)
- User list, profile creation, updating, and deletion

## How to run
### Backend
1. Open the `/backend` directory.
2. Run `npm install` to install server dependencies.
3. Run Mongoose/SQL server and start node via `node server.js` or `npm run dev`.

### Frontend
1. Open the `/frontend` directory.
2. Run `npm install` to install React dependencies.
3. Run `npm run dev` to start the frontend server.
